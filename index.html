Job Tracker API

A professional backend Job Application Tracking System built using FastAPI, SQLAlchemy, SQLite, JWT Authentication, and external job API integration.

View GitHub Swagger Docs

Project Overview

Job Tracker API helps users manage their complete job application process through secure REST APIs. Users can register, login, manage companies, track job applications, schedule interviews, save job preferences, receive real-time job recommendations, save recommended jobs, and view dashboard analytics.

Key Features

Authentication

Secure registration and login using JWT authentication and password hashing.

Job Tracking

Create, view, update, and delete job applications with status, salary, location, and work model.

Interview Management

Schedule and manage interviews with interview type, date, round number, and duration.

Job Recommendations

Uses user preferences and an external job API to fetch real-time job recommendations.

Dashboard Analytics

Shows total jobs, companies, interviews, preferences, and success rate.

Technologies Used

  • Python
  • FastAPI
  • SQLAlchemy ORM
  • SQLite
  • Pydantic
  • JWT Authentication
  • Requests Library
  • Swagger UI

Main API Endpoints

  • POST /register - Register a new user
  • POST /login - Login and receive access token
  • POST /companies - Add company
  • GET /jobs - Get jobs with pagination and filters
  • POST /interviews - Schedule interview
  • POST /job-preferences - Save job preferences
  • GET /recommended-jobs - Fetch real job recommendations
  • POST /save-recommended-job - Save recommended job
  • GET /dashboard - View analytics dashboard

Development Process

The project was first developed in a single main.py file to quickly test the core APIs. It was then refactored into a modular FastAPI structure using separate files for routes, schemas, models, authentication, database configuration, and external job API logic.
